Instalar Mods

Notas Importantes

  • Compatibilidad: No requiere Minecraft Premium
  • Sincronización: Todos deben tener los mismos mods
  • Versión: Usar exactamente Forge 47.4.0
  • Rendimiento: Cerrar aplicaciones innecesarias

🔧 Solución de Problemas

Si experimentas crashes o problemas de conexión:

  • Verifica que Java 17 esté instalado correctamente
  • Confirma que todos los mods estén en la carpeta correcta
  • Asegúrate de que ZeroTier esté conectado
  • Reinicia el launcher después de instalar los mods
